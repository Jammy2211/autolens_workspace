Subhalo (Group)
===============

This folder contains scripts demonstrating dark matter subhalo detection in group-scale strong lenses.

Files
-----

- ``simulator.py``: Simulates a group-scale lens dataset that includes a dark matter subhalo perturbing
  the lensed source emission.
- ``detect/start_here.py``: Runs the full SLaM + SUBHALO pipeline for detecting a dark matter subhalo
  in a group-scale lens, including grid search and refinement.
- ``detect/database.py``: Shows how to query the database of subhalo detection results for group-scale
  lenses.
