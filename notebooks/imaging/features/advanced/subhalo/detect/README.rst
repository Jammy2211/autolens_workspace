The ``imaging/advanced/subhalo/detect`` folder contains example scripts for strong lens dark matter subhalo
detection.

Files
-----

- ``start_here``: An example performing dark matter subhalo detection using the SLaM pipelines.
- ``database``: Dedicated database functions for dark matter subhalo detection.- ``examples``: Dark matter subhalo detection using different types of lens model (e.g. light profile sources).
