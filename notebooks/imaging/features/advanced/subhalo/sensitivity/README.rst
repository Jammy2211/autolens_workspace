The ``imaging/advanced/subhalo/sensitivity`` folder contains example scripts for strong lens dark matter subhalo
sensitivity mapping.

Files
-----

- ``start_here``: An example performing dark matter subhalo sensitivity analysis.
- ``database``: Dedicated database functions for dark matter subhalo sensitivity analysis.
- ``slam_source_pixelized``: Dark matter subhalo sensitivity mapping using pixelized sources using the SLaM pipelines.
- ``slam_source_parametric``: Dark matter subhalo sensitivity mapping using a light profile source using the SLaM pipelines.
