The ``database`` folder contains tutorials explaining how to use a SQLite3 database for managing large
suits of  modeling results.

Other results examples load results by scraping the results folders in the ``output`` folder, which is efficient
for small results libraries but scales poorly. An SQLite3 database can be built to speed this up up for large samples.

Files
-----

- ``start_here``: An overview of using the sqlite3 database tools.
