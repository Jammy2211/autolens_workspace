Subhalo Detection (Group)
=========================

This folder contains scripts for detecting dark matter subhalos in group-scale strong lens data.

Files
-----

- ``start_here.py``: The main script that runs the full SLaM + SUBHALO pipeline for group-scale lenses.
  This includes the SOURCE LP, SOURCE PIX, LIGHT LP, MASS TOTAL and SUBHALO pipelines, all adapted for
  group-scale lens modeling with main lens galaxies and extra galaxies.
- ``database.py``: Shows how to query the database of subhalo detection results, including extracting
  log evidence comparisons and visualizing grid search results.
