The ``examples` folder contains scripts which show how to use **PyAutoGalaxy** built-in visualization tools.

The API provides a simple interface with matplotlib for making plots that does not require the user to
write any matplotlib code themselves.

This is illustrated in the ``overview`` / ``overview.ipynb`` file, where new users should begin.

Files
-----

- ``mat_plot``: Objects which wrap Matplotlib functions which customize a figure's appearance.
- ``plotters``: Plot specific **PyAutoLens** objects (e.g. ``Tracer``).
- ``searches``: Visualization tools for the results of a non-linear search (e.g. ``corner_anesthetic``).
- ``visuals``: Include visuals on a plot (e.g. a critical curve on an image).
