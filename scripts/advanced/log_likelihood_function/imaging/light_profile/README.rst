The ``log_likelihood_function/light profile`` folder contains example scripts showing step-by-step visual guides
to how likelihoods are evaluated using a light profile.

Files (Advanced)
----------------

- ``log_likelihood_function.py``: A step-by-step visual guide to how a likelihood is evaluated using a light profile.
- ``contributor_guide.py``: GitHub URL links to every package, module and function called to perform likelihood evaluations to help contributors.
